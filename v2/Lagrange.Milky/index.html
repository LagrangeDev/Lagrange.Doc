Lagrange.Milky ​

Lagrange.Milky 是 Milky 协议的实现，可用于和 NoneBot 等机器人框架进行集成。

获取 ​

目前 Lagrange.Milky 尚未发布 Release，尝鲜构建版本可从 Actions 获取。使用过程中出现 Bug 请及时反馈。

Lagrange.Milky 使用了 Native AOT 技术，无需 .NET 运行时，目前支持的平台如下：

  • Windows x64
  • Linux x64
