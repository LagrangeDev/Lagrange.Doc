Lagrange.Core ​

快速开始 ​

  1. 使用以下任一方案引用 Lagrange.Core
    • 从 Nuget 导入 Lagrange.Core 到你的项目 (版本可能不为最新版本)
    • 通过 git submodule 功能将 Lagrange.Core 作为解决方案中的项目, 并在你的项目中引用
  2. 创建 Bot 实例
  3. 登录
  4. 监听并处理 Bot 收到的消息等事件
  5. 发送消息
